Automatic Labelers

Packaging labeling equipment

Designed and manufactured in Poland. Attractive pricing. Built for labeling bottles, tubes, jars and more. The machine delivers high labeling precision, full process repeatability, and the ability to run in line with a printer.

Machine capabilities

The machine can additionally be equipped with an InkJet-type date printer.

The labeler allows very precise adjustment of head speed, gluing angles, conveyor speed, wrap-around unit and boxer unit.

All components used to build the machine come from reputable suppliers — this is not a low-quality product imported from China.

Line integration

The machine can be combined into a single production line — by adding a filler, a capper, a cap/cork feeder, and infeed, buffer and outfeed tables.

From 56,000 PLN net

Specification

  • Conveyor dimensions: width 25 cm, belt width 20 cm, height with base 85 cm, length 170 cm
  • Max. diameter of labeled object: 20 cm
